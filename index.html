C# Cheat Sheet

Welcome to my C# cheat sheet
This sheet is still in progress and will be updated as soon as possible.

Data Types

Value Types:
int, char, float, double, bool, byte, short, long, decimal
Reference Types:
string, class, array, interface, delegate

Variables

In C#, variable names are written in camelCase:
int num = 999;
string word = "Hello World";
bool isTrue = true;
double price = 19.99;

Basic Syntax

Creating a class:
public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}
Creating a method:
public void SayHello()
{
    Console.WriteLine("Hello!");
}

Control Structures

Conditional Statements:
if (condition)
{
    // code to execute if condition is true
}
else
{
    // code to execute if condition is false
}
Loops:
For Loop:
for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}
While Loop:
int i = 0;
while (i < 10)
{
    Console.WriteLine(i);
    i++;
}

Methods

Creating a method with parameters:
public int AddNumbers(int num1, int num2)
{
    return num1 + num2;
}
Calling a method:
int result = AddNumbers(5, 10);
Console.WriteLine(result); // Outputs: 15

Properties

Creating a property:
public class Car
{
    public string Color { get; set; }
    public string Model { get; set; }
}
Using a property:
Car myCar = new Car();
myCar.Color = "Red";
myCar.Model = "Toyota";
Console.WriteLine(myCar.Color); // Outputs: Red

Arrays

Declaring and initializing an array:
int[] numbers = new int[] {1, 2, 3, 4, 5};
string[] words = {"Hello", "World"};
Accessing array elements:
int firstNumber = numbers[0];
Console.WriteLine(firstNumber); // Outputs: 1
string firstWord = words[0];
Console.WriteLine(firstWord); // Outputs: Hello

Namespaces

Using a namespace:
using System;
using System.Collections.Generic;
Creating a namespace:
namespace MyNamespace
{
    public class MyClass
    {
        public void MyMethod()
        {
            // Method code here
        }
    }
}
